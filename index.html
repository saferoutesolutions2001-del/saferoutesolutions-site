Privacy Policy - JRM Utility

Effective Date: February 2026

JRM Utility is developed by Safe Route Solutions.

This application does not require user registration and does not collect personal information such as name, email address, phone number, or location.

Camera Usage

The application uses the device camera strictly for scanning QR codes required to retrieve Journey Risk Management (JRM) reports. No images or video are stored or transmitted.

Local Storage

The application stores the selected Depot ID and downloaded JRM files locally on the user's device for offline access. These files remain within the application's private storage and are not shared externally.

Cloud Storage and Technical Data

The application retrieves pre-generated JRM documents from secure cloud storage (Firebase Storage). Standard technical information such as device identifiers may be processed automatically by Firebase services for secure file delivery. This information is not used to personally identify users.

Data Sharing

The application does not sell, trade, or share personal data with third parties.

Security

All document retrieval is performed via encrypted HTTPS connections.

User Control

Users may clear application data or uninstall the app to remove locally stored information.
